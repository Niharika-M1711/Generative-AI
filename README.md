# Generative-AI
Generative AI projects covering LLM agents, automation pipelines and conversational bots using LangChain and Groq


# Generative AI Projects

This is where everything came together. After building ML models and deep 
learning architectures, GenAI felt like the natural next step — moving from 
models that predict to systems that think, plan and act.

These projects explore what happens when you give LLMs real tools — 
web access, email, APIs — and let them do actual work.

## Projects

| Project | Description | Tech |
|---------|-------------|------|
| Briefing Bot | AI agent that reads and summarizes daily news | LangChain, Groq |
| Email Automation | Automated email generation and sending pipeline | LangChain, Gmail API |
| Travel Planner Bot | Conversational travel itinerary generator | LangChain, Groq, LLaMA-3 |

## What I learned

The jump from "calling an API" to "building an agent" is bigger than it sounds. 
Agents need to handle failures, decide between tools, maintain context and 
produce consistent outputs — none of which is easy.

Email automation taught me how much prompt engineering actually matters. 
The same model with a different prompt produces completely different results.
